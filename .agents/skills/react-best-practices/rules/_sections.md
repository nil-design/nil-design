# React Rule Sections

1. Eliminating waterfalls: async rules for avoiding sequential latency.
2. Bundle size: imports, dynamic loading, and analyzable module paths.
3. Client-side data and effects: listeners, storage, and client request behavior.
4. Re-render optimization: render derivation, memo use, stable callbacks, and hook splitting.
5. Rendering performance: DOM, layout, SVG, hydration, and resource hints.
6. JavaScript performance: lower-level data structure and loop choices.
7. Advanced React patterns: latest refs, handler refs, and one-time initialization.
