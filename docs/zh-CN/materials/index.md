# {{ $frontmatter.title }} <Badge type="warning" text="WIP" />

`@nild/materials` 当前仅作为规划入口保留，短期不提供可用能力，也不建议安装或在生产环境中依赖。

现阶段可以优先使用这些已经可用的包：

- `@nild/components`：基础与交互组件。
- `@nild/hooks`：React Hooks。
- `@nild/icons`：图标能力。
- `@nild/i18n`：国际化能力。
- `@nild/shared`：基础类型、工具函数与 React 基础能力。

未来的 `@nild/materials` 会定位为更高层的组合物料，基于当前基础能力沉淀更完整的场景组合。
