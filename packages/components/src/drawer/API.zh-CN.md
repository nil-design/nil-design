### Drawer Props

> 继承自 `Omit<ModalProps, 'placement'>`

| 属性名 | 描述 | 类型 | 默认值 |
| :-- | :-- | :-- | :-- |
| placement | - | `'left' \| 'right' \| 'top' \| 'bottom'` | - |
