### Select Props

> 等价于 `SingleSelectProps | MultipleSelectProps`

### Select.Option Props

> 继承自 `Omit<HTMLAttributes<HTMLDivElement>, 'children' | 'value'>`

| 属性名 | 描述 | 类型 | 默认值 |
| :-- | :-- | :-- | :-- |
| value* | - | `T` | - |
| label* | - | `string` | - |
| size | - | `InputSize` | - |
| disabled | - | `boolean` | - |
| selected | - | `boolean` | - |
| active | - | `boolean` | - |
| children | - | `ReactNode` | - |
